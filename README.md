# Capstone-Project-Data-Analytics-Program
NLP-Based Research Paper Analysis 
Capstone Project by Aura Putri Mahabah (2210312024)

## Deskripsi
Aplikasi GUI berbasis Python untuk analisis paper penelitian menggunakan NLP dan integrasi IBM Granite untuk klasifikasi dan ringkasan otomatis.

## Fitur
- Import data dari Zotero/Mendeley
- Analisis Sentimen dan Keyword Extraction
- Integrasi IBM Granite
- Dashboard pastel mint
- Export hasil analisis
